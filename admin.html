Generated JSON Output

Instructions:

  1. Fill out the form for either a blog post or a news article.
  2. Click the "Generate ... JSON" button.
  3. Copy the entire content from the "Generated JSON Output" textarea above.
  4. Open the respective JSON file in your project:
    • For blog posts: `claudconverter-pro/data/blog-posts.json`
    • For news articles: `claudconverter-pro/data/news-articles.json`
  5. These files contain a JSON array (starts with `[` and ends with `]`).
    • To add a new item: Paste the copied JSON object inside the array. If the array is not empty, add a comma (`,`) after the preceding object before pasting. Ensure the new object is also followed by a comma if it's not the last item in the array.
    • To update an existing item: Find the object in the array that has the same `id` as the one you are updating. Replace that entire object (from its opening `{` to its closing `}`) with the copied JSON.
  6. Save the JSON file. The changes should reflect on your blog/news pages after a page refresh (assuming the site is re-fetching the JSON).
